# Alarm-Clock
Alarm Clock is maked with arduino
Features:
1- Can show time on lcd screen
2- Can hold correct time for +5 years with RTC module
3- Can create new alarms
4- Can delete alarms
5- Can change alarm times
Used Parts:
1-Arduino Uno
2-Lcd Screen
3-RTC
4-4x4 matrix keypad
5-Buzzer
6-9v battery
